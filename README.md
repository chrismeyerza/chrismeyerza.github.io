# chrismeyerza.github.io

Personal site. Plain HTML and CSS, hosted on GitHub Pages.

## Files

```
├── index.html     — Home
├── about.html     — Background
├── projects.html  — Placeholder
├── contact.html   — Contact details
└── styles.css     — Shared styles
```

## Editing

Open the folder in VS Code, edit any HTML file, save, commit, push. GitHub Pages auto-deploys within a minute.

To preview locally: right-click any `.html` file in VS Code → **Open with Live Server**.
